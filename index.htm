Memory
Gra memory to popularna gra karciana, która sprawdza pamięć graczy i umiejętność zapamiętywania.
Celem gry Memory jest odkrycie jak największej liczby par identycznych kart.Gra rozwija zdolności pamięciowe, spostrzegawczość i strategiczne myślenie.
Szybka instalacja:
  1. Pobierz tą aplikacje zgodne z twoim systemem:
    2. memory.exe dla środowiska windows
    memory dla środowiska linux
  2. Otwórz pobrany plik memory
  3. Odblokuj plik w zaporze antywirusowej
Zasady Gry:
  • Celem gry jest odkrycie jak największej liczby pasujących par kart. Gracz, który zdobędzie więcej punktów (znajdzie więcej par), wygrywa grę.
    1. Rozpoczęcie nowej gry: (opcja 1)
      • Gracze nadają sobie pseudonimy.
      • Ustalany jest rozmiar kart (w formacie n×n), ciąg znaków używanych do budowy kart oraz liczbę kart do odkrycia (będzie ich 2 razy więcej bo występują parami).
      • Karty są generowane z losowo wybranymi znakami w zadanych rozmiarach i zapisywane do pliku z którego mogą być odczytywane jeśli rozgrywka z jakiegoś powodu się nie zakończy
      • Punkty oraz kogo kolej widoczne są pod kartami
      • Gracze wprowadzając na zmianę numery kart (znajdujące się na górze kart) do odkrycia
      • Jeśli karty się zgadzają to dany gracz otrzymuje punkt w przeciwnym wypadku kolej przysługuje dla 2 gracza
      • Jak wszystkie karty zostaną odkryte następuje koniec gry oraz wyłonienie zwycieżców
    2. Wczytanie ostatnio zapisanej gry:
      3. Istnieje możliwość odtworzenia ostatniej nie zakończonej rozgrywki (opcja 2)
    3. Domyślna gra: (opcja 3)
      4. Domyślna opcja jest dla tych którzy nie mają czasu i chcą zagrać szybką partie
